# 🧪 مختبر المشاريع | Project Lab

**من فكرتك إلى شريكك إلى عقدك – إلى لوحة تحكم مشروعك**

أداة مفتوحة المصدر بالعربية، صُممت لمساعدة المبرمجين ورواد الأعمال التقنيين في العالم العربي على تقييم أفكارهم، واختيار الشركاء المناسبين، وتوثيق اتفاقهم، وإدارة مشاريعهم بشكل عملي ومباشر.

---

## 📂 هيكل المستودع

| الملف | الوصف |
|---|---|
| `index.html` | **صفحة المقارنة** – تتيح لك الاختيار بين الإصدارين |
| `v1.html` | **الإصدار الأساسي** – المسار الثلاثي (فكرة ← شريك ← ميثاق) |
| `v2.html` | **الإصدار المتكامل** – المسار الرباعي مع لوحة تحكم المشروع |

---

## 🔹 الإصدار الأساسي (`v1.html`)

المسار الثلاثي الكلاسيكي:

1. **اختبر فكرتك** – 4 أسئلة واقعية مستوحاة من منهجيات Lean Startup
2. **قيّم شريكك** – 4 معايير لتقييم الشريك المحتمل
3. **ولّد ميثاق شراكة** – عقد أخلاقي بسيط جاهز للنسخ

---

## 🚀 الإصدار المتكامل (`v2.html`)

كل ما في الإصدار الأساسي، بالإضافة إلى:

4. **لوحة تحكم المشروع**
   - إدارة أعضاء الفريق (إضافة/حذف)
   - تتبع المهام لكل عضو مع نسب الإنجاز
   - شريط تقدم عام للمشروع
   - **نظام طوارئ 🚨** – طلب مساعدة أو استبدال عضو وتوليد منشور جاهز للنشر في مجتمعك
   - مولد منشورات الجروب – تحديثات الحالة جاهزة للنسخ

---

## 🎯 المميزات العامة

- 🇸🇦 بالعربية بالكامل
- 🧩 تصميم خطي بسيط (Wizard)
- 💾 حفظ تلقائي للبيانات في متصفحك (`localStorage`)
- 📋 نسخ بنقرة واحدة للمواثيق والمنشورات
- 🌐 يعمل بدون سيرفر أو قاعدة بيانات – ملف HTML واحد لكل إصدار
- 🆓 مجاني ومفتوح المصدر (MIT)
